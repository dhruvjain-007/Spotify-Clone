# Spotify Clone

A modern Spotify inspired music player built using HTML, Tailwind CSS, and Vanilla JavaScript. This project replicates the look and feel of the latest Spotify web app with responsive design and interactive music playback features.

---

## Features

* Play, pause, next, previous controls
* Interactive progress bar
* Volume control
* Fully responsive (desktop, tablet, mobile)
* Modern Spotify-like UI (dark theme)
* Dynamic playlists and song list
* Search functionality (optional)
* Like/Favorite songs (optional)

---

## Tech Stack

* HTML5
* Tailwind CSS
* Vanilla JavaScript
* HTML5 Audio API

## Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/your-username/spotify-clone.git
```

2. Navigate to the project folder:

```bash
cd spotify-clone
```

3. Open `index.html` in your browser
   (or use Live Server in VS Code)

---

## How It Works

* Songs are loaded from a mock dataset (JSON/JS array)
* JavaScript controls playback using the Audio API
* UI updates dynamically based on user interaction
